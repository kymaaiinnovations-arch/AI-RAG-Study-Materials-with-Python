# Chapter 03 - Layouts

This chapter explains how to organize a Streamlit app using layouts and containers.

## What this app demonstrates

- Two-column layout
- Sidebar content
- Expanders
- Markdown sections
- Images inside the app
- Buttons and selection controls

## Main concepts

- Column layout
- Sidebar usage
- Expander blocks
- Displaying formatted content with Markdown

## Purpose

This app demonstrates how to make a Streamlit page more structured and visually appealing for users.
