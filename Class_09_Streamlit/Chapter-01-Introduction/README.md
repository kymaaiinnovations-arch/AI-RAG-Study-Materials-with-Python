# Chapter 01 - Introduction

This chapter introduces the basic structure of a Streamlit app.

## What this app demonstrates

- Creating a title and subtitle
- Displaying text and messages
- Using a select box for user input
- Showing a success message after selection

## Main concepts

- Streamlit basics
- Simple text and display elements
- Interactive widgets
- User input handling

## Purpose

This app is a beginner-friendly example of how to build an interactive web app with Streamlit.
