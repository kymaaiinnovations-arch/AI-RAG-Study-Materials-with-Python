# Chapter 04 - Working with Data

This chapter shows how to work with uploaded CSV data in a Streamlit app.

## What this app demonstrates

- Uploading a CSV file
- Reading the file with pandas
- Showing a preview of the dataset
- Displaying dataset statistics
- Filtering rows by city

## Main concepts

- File upload in Streamlit
- DataFrames with pandas
- Data exploration and filtering
- Building a simple dashboard

## Purpose

This app helps learners understand how to build data-driven applications using Streamlit and pandas.
