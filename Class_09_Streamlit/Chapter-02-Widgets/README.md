# Chapter 02 - Widgets

This chapter focuses on Streamlit widgets and how they collect user input.

## What this app demonstrates

- Buttons
- Checkboxes
- Radio buttons
- Select boxes
- Sliders
- Number inputs
- Text inputs
- Date inputs

## Main concepts

- Interactive user input
- Displaying widget values dynamically
- Creating simple forms in Streamlit

## Purpose

This app shows how different widgets can be combined to create a registration-style interface for an AI learning platform.
