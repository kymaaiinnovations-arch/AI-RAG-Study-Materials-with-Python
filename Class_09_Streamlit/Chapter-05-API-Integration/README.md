# Chapter 05 - API Integration

This chapter demonstrates how to connect a Streamlit app with an external API.

## What this app demonstrates

- Accepting user input for an amount
- Selecting a target currency
- Calling a live exchange rate API
- Displaying converted results
- Handling API errors gracefully

## Main concepts

- API requests with requests
- Working with JSON responses
- Displaying results in the UI
- Error handling in web apps

## Purpose

This app shows how to create a simple live currency converter using Streamlit and a public API.
